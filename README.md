##Useful commands

List outdated bower packages:

    bower list
    
Update bower after modifying `bower.json`:

    bower update

List outdated npm packages:

    npm outdated
    
Or more readable with:

    npm-check-updates
    
To automatically update `package.json`:

    npm-check-updates -u

Update npm packages according to `package.json`:

    npm update
